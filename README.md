# CarouselImage

功能介绍
-------------------------------------
A Image detail show widget 一个图片查看插件<br/>
用于手机端图片放大全屏查看，有点击切换和手势划动切换两种切换方式<br/>
使用时按照ul li img 的层次格式写即可，后期有时间会扩展成自定义层次支持<br/>
主要支持微信手机端，会自动按照大小居中显示<br/>
